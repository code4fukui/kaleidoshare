# Kaleidoshare

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Let's create and share kaleidoscopes on the web.

## Features
- Create and share kaleidoscope designs
- Easily configure settings like shape, color, and animation
- View and interact with other users' creations

## Requirements
- Node >= v20
- Deno >= v1.32 (for KV)

## Usage

### Install dependencies
```
npm ci
npx playwright install --with-deps
```

### Develop
```
npm run dev
```

### Build
```
npm run build
```

### Test
```
npm test
```
